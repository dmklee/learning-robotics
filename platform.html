Capable and Low-Cost Platform

At about $250 per kit, exploring topics in robotic manipulation has never been more accessible. Despite the low price point, the robot kit is highly capable, offering accurate positional control, five degrees of freedom, and visual feedback. The arm is modeled in a virtual simulator, providing more advanced features like inverse kinematics and obstacle avoidance.

Robot Kit

The robot kit is built around the Hiwonder xArm, which provides 5-DoF grasping capabilities. Coupled with a camera to locate nearby objects, the learning opportunities are endless. With our Python library that exposes high level motion primitives, complex manipulation tasks can be performed with only a few lines.

Simulator

The simulator is built using Pybullet to accurately represent the appearance and kinematics of the real robot. Using the simulator is a great way to test out the platform before purchasing and could serve the basis for a virtual educational program.
